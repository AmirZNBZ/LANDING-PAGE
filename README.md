### Installation

```bash
git clone https://github.com/AmirZNBZ/LANDING-PAGE.git
cd LANDING-PAGE
npm install
```

### Running Locally

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
